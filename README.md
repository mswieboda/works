# works

[Factorio](https://factorio.com/) clone using [Crystal](https://crystal-lang.org/) and [Allegro](https://liballeg.org/)

## Installation

```
make
```

or

```
make test
```

to run the quickly compiled, more debuggable build, or

```
make release
```

for the more efficient production ready build.
